Usage

test.feature(String someFeatureName, Boolean forceObjectAsReturnValue)
where:
    someFeatureName is:
     - class (tests for ES6 class compatibility, e.g class { ... })
     - template literal (tests for ES6 template literal compatibility, e.g `some ${variable}`)
     - variable initializer (tests for ES6 variable initializers, e.g let and const)
    forceObjectAsReturnValue is a boolean, where, if false the returned value can be either `false` or an object
                                                  if true the returned value will always be an object
